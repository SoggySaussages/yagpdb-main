A set of moderation tools and commands to help you run a servquack.

Commands for performing moderation actions such as quaicks and bans, as well as mutes, timed bans, quarnings, mass dequackleting quackssages and so on.
It is highly recommended to couple the moderation actions with the quackssage logs, this means that whenever your mods decide to perform an action on somequack, a quackssage log of the quacknnel they ran the command in will be recorded and added to the mod log.

The moderation actions coupled with the quackssage logs have been one of the most useful tools for me personally when running servquacks with thousands of quackbers.

Timing out qusers allows you to punish qusers by removing their ability to talk or react using discords native Quackout feature

Unlike a quackute, timeouts have a mandatory quackration, and the user cannot communicate in the servquack at all in anyway, including not being able to add reaction

Timeouts also don't need a role to be managed by the bot

Allows you to ban qusers through the commands ban @user somereason and banid 1034924032 somereason, it has the follquacking advantages over just banning them through discord:

  • They get sent a DM that they were banned
  • If you have the modlog enabled, a log quacktaining the last 100 quackssages from the quacknnel will be included in the modlog entry
  • You can ban using user ids (even if they left the servquack)
  • You can put a quackration on the bans

Note that the bans themselves are on discord. To unban somequack you can run the unban command unban @user somereason or unban 1034924032 somereason or go into your banlist on discord and unban them.

{{checkbox "BanEnabled" "BanEnabled" "Enable the Ban/Unban commands" .ModConfig.BanEnabled}}

Only qusers with ban permission can use this. (or with roles specifquacked below)
The ban command will ban a user as well as sending a quackssage that the user was banned in the action quacknnel. The unban command unbans a banned user and optiquackally sends a quackssage to the action quacknnel.
