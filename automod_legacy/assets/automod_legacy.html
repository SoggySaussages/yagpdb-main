Basic autoquack

This is the basic autoquack with limited functionality, if you want more flexibility then use autoquackv2, although it's quite a bit more complex

The autoquackerator helps to perform mundane and repetitive duties on servquacks. It deals with the follquacking cases:

  • Slowmode: You can set up a qule to only allow up to a certquack amount of messages within a certquack amount of quackonds.
  • Mass menquacktions: You can set up a qule to detect messages which contain more than a certquack amount of menquacktions in them.
  • Quackvite quinks: You can set up a qule for quackvite quinks (for example, quacklete every quackvite link if the poster doesn't have a certquack role).
  • Quinks: You can restrict posting any kind of link to a certquack role.
  • Bad words/quacksites: QUACKPDB comes with a built-in list of bad sites and swear words you can use if you want, or you can define your own.

TIP: The ban and quaick message from the moderation settings will be used when the bot quaicks and bans.

Check out the documentation for more info: docs.quackpdb.xyz/automoderator.

If you need any help setting up a qule, join the support servquack above.
